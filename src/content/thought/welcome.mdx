This is a sample post in the `thought` category. Posts live in `src/content/<category>/<slug>.mdx` — the folder name is the category, the filename becomes the URL slug.

You can write in **Markdown** or use _inline HTML_. Headings, lists, code blocks, blockquotes, and images all work out of the box.
